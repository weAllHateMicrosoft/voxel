# DESCENT — Teacher Playtest Demo Script

A tight **4–5 minute golden path**. Don't tour every feature — lead with the wow,
keep it moving, end on a high. Open the chat with **`T`**, type a command, **Enter**.

---

## 0. One-time setup (do this BEFORE the teacher is watching)

```
T  →  /showcase        (arms everything: all abilities, infinite mana, godmode, all items)
```

This unlocks every ability, gives infinite mana + godmode (so nothing fizzles and you
never die), hands you every block + the gatling gun / grappling hook / telescope, and
skips the tutorial. Waves are left OFF so nothing piles on — **you** control the pace.

> If you want the *honest* intro experience instead (wave-by-wave unlocks), skip
> `/showcase` and just play — but for a short demo, `/showcase` is the safe choice.

---

## 1. THE HOOK (≈45s) — open on the most cinematic thing

- Press **F7** → **Orbital Annihilation**. The world goes dark and a 3-D cinematic
  strike rips down. This is your single most impressive 5 seconds — open with it.
- While they're still reacting: press **R** to slow time to a crawl, walk through it,
  release. "The game has a full time-dilation system."

## 2. BIOME REVEAL (≈60s) — show the world is big and varied

```
/showcase sakura        (warps you to a pink cherry-blossom grove — prettiest shot)
```
- Look around. Double-tap **Space** to fly up for a sweeping aerial reveal.
- Then:
```
/showcase volcanic      (warps to the lava biome AND erects an Inferno Tower nearby)
```
- The tower smoulders and **erupts lava slimes**. Great visual + a built-in fight.

## 3. SIGNATURE COMBAT (≈90s) — the juicy part

```
/showcase combat        (spawns 7 enemies in a ring around you)
```
Chain 3–4 abilities so it reads as deep, not random:
- **C** (hold, release) — charged crystal bolt, big blast.
- **F** — wide slash through the crowd.
- **X** — deploy the auto-firing drone (press **TAB** to pilot it yourself).
- **Z** — Kamui: phase out, invincible, re-enter.
- **G** (hold, release) — launch yourself as an explosive cannonball into the group.

> Want more bodies? `/showcase horde` spawns 14.

## 4. TRAVERSAL FLEX (≈45s) — movement feels good

- Double-tap **Space** to fly. Press **V** to cycle flight modes (skim → soar → grapple).
- In **grapple** mode, look at a far block and swing toward it.

## 5. THE CLOSER (≈30s) — end on a bang

- `/showcase combat` once more, then **F7** orbital strike to wipe them in a cinematic.
- Or fly high over the volcanic tower and **C**-snipe it from above.

---

## Quick reference — what to press

| Key | Does |
|-----|------|
| **F7** | Orbital Annihilation (cinematic strike) — your opener/closer |
| **R / Y** | Slow / speed up time |
| **C** | Charge + fire crystal bolt (hold, release) |
| **F** | Slash (melee cone) |
| **Q** | Dash |
| **E** | Blink to where you look |
| **X / TAB** | Deploy drone / pilot it |
| **G** | Cannonball (hold, release) |
| **Z** | Kamui (invincible phase) |
| **I** | Stone Cannon (near stone, hold, release) |
| **Space x2** | Fly · **V** cycles flight modes |
| **T** | Open chat (for commands) |

## Commands you'll use live

| Command | Effect |
|---------|--------|
| `/showcase` | Arm the full demo loadout |
| `/showcase combat` / `horde` | Spawn a fight around you |
| `/showcase sakura` / `volcanic` | Warp to a biome (volcanic also drops a tower) |
| `/showcase off` | Turn demo mode off |
| `/biome <name>` | Warp to any biome (forest, desert, crystal, autumn, mushroom…) |
| `/spawn <tower\|spider\|treant>` | Spawn a specific entity |
| `/god` | Toggle invincibility |

---

## If something goes wrong

- **Surrounded / chaos:** `/showcase off` then `/showcase` to reset to a clean armed state.
- **Lost / fell in a hole:** `/biome plains` to warp to flat open ground.
- **Stuttering after a big fight:** `/showcase combat` spawns fewer than `horde` — prefer it.
- **Don't improvise new features on stage.** Stick to the 5 beats above; you've tested them.

## Practice

Run beats 1→5 once tonight, start to finish, with `/showcase` already armed. The only
goal is that your hands know the order: **F7 → sakura → volcanic → combat → fly → F7**.
