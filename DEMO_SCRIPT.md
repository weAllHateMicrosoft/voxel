# DESCENT — Demo Script (`/showcase` quick tour)

> **Note:** pressing **PLAY** now starts the real **guided progression** — a tutorial, then
> a new power earned every wave, climaxing in the god-tier weapons (Gatling → Radar → Orbital
> → Time Stop → Flight). Let the teacher play *that* for the intended "teach & reward" arc.
> This script is the **fast `/showcase` tour** for when *you* want to show off everything at
> once without playing through all the waves.

A tight **4–5 minute golden path** using `/showcase` (everything unlocked, slots 1-5 armed).

---

## 0. One-time setup (do this BEFORE the teacher is watching)

```
T  →  /showcase        (arms everything: all abilities, infinite mana, godmode)
```

All 4 ability weapons land in hotbar slots **2–5**. Select any and press **RMB** to fire.
A hint line appears at the bottom of the screen showing exactly what each weapon does.

> If you want the honest wave-by-wave experience, skip `/showcase` — but for a short demo
> this is the safe choice.

---

## 1. THE HOOK (≈45 s) — open on the most cinematic thing

- Select hotbar **slot 3** (Orbital Annihilation) → press **RMB**.
  The world goes dark and a 3D cinematic laser strike rips down.
  *Also works with* **F7** if you forget.
- While they're still reacting: press **R** to slow time to a crawl, walk through it,
  release. *"The game has a full time-dilation system."*

## 2. BIOME REVEAL (≈60 s) — show the world is big and varied

```
/showcase sakura        (warps to a pink cherry-blossom grove)
```
- Look around. Double-tap **Space** to fly up for a sweeping aerial reveal.
- Then:
```
/showcase volcanic      (warps to the lava biome + erects an Inferno Tower)
```
- The tower smoulders and erupts lava slimes. Great visual + a built-in fight.

## 3. SIGNATURE COMBAT (≈90 s) — the juicy part

```
/showcase combat        (spawns 7 enemies in a ring around you)
```
Chain 3–4 weapons so it reads as deep, not random:
- **Slot 2** (Sniper) — hold RMB, release: big crystal blast.
- **F** — wide slash through the crowd.
- **X** — deploy the auto-firing drone (press **TAB** to pilot it yourself).
- **Z** — Kamui: phase out, invincible, re-enter.
- **G** (hold, release) — launch yourself as an explosive cannonball into the group.

> Want more bodies? `/showcase horde` spawns 14.

## 4. TRAVERSAL FLEX (≈45 s) — movement feels good

- Double-tap **Space** to fly. Press **V** to cycle flight modes (skim → soar → grapple).
- In **grapple** mode, look at a far block and swing toward it.

## 5. THE CLOSER (≈30 s) — end on a bang

- `/showcase combat` once more, then **Slot 3** orbital strike to wipe them in a cinematic.
- Or fly high over the volcanic tower and **Slot 2** snipe it from above.

## Quick reference — what to press

| Key | Does |
|-----|------|
| **Slot 1 + LMB** | Gatling Gun — rapid fire |
| **Slot 2 + RMB** | Sniper — hold to charge, release fires crystal bolt |
| **Slot 3 + RMB** | Orbital Annihilation (also **F7**) |
| **Slot 4 + RMB** | Time Domain — freeze all enemies (also **F8**) |
| **Slot 5 + RMB** | Stone Cannon — hold near stone, release fires boulder |
| **R / Y** | Slow / speed up time |
| **F** | Slash (melee cone) |
| **Q** | Dash |
| **E** | Blink to where you look |
| **X / TAB** | Deploy drone / pilot it |
| **G** | Cannonball (hold, release) |
| **Z** | Kamui (invincible phase) |
| **Space x2** | Fly · **V** cycles flight modes |
| **T** | Open chat (for commands) |
| **Left Alt** | Open backpack (reassign weapons to hotbar) |

## Commands you'll use live

| Command | Effect |
|---------|--------|
| `/showcase` | Arm the full demo loadout |
| `/showcase combat` / `horde` | Spawn a fight around you |
| `/showcase sakura` / `volcanic` | Warp to a biome (volcanic also drops a tower) |
| `/showcase off` | Turn demo mode off |
| `/biome <name>` | Warp to any biome (forest, desert, crystal, autumn, mushroom…) |
| `/spawn treant` | Spawn a disguised tree-enemy |
| `/spawn spider` | Spawn a procedural IK spider |
| `/god` | Toggle invincibility |

## If something goes wrong

- **Teacher asks what to press:** hint text shows at the bottom whenever a weapon is selected.
  Just say *"select a slot with 1–5 and right-click"*.
- **Surrounded / chaos:** `/showcase off` then `/showcase` to reset to a clean armed state.
- **Lost / fell in a hole:** `/biome plains` to warp to flat open ground.
- **Stuttering after a big fight:** `/showcase combat` spawns fewer than `horde` — prefer it.
- **Don't improvise new features on stage.** Stick to the 5 beats above; you've tested them.

## Practice

Run beats 1→5 once tonight, start to finish, with `/showcase` already armed.
The only goal is that your hands know the order: **Slot 3 → sakura → volcanic → combat → fly → Slot 3**.
